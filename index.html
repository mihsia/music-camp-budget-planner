取得 Client ID 步驟

  1. 開啟 Google Cloud Console 憑證頁，選擇或建立一個專案。
  2. 到 Google Drive API 頁面，確認已啟用。
  3. 若系統要求設定 Google Auth Platform，先完成應用程式名稱、使用者支援信箱與 Test users。
  4. 點選「建立憑證」或「Create client」，類型選擇「Web application」。
  5. 在「Authorized JavaScript origins」加入本系統網址。
  6. 建立後複製「Client ID」，貼回上方欄位，再按左側的「連接 Google Drive」。

本系統要填的來源網址

正式使用請填：

https://mihsia.github.io

若在本機測試，另加：

http://localhost:8765

若網址列顯示 127.0.0.1，另加：

http://127.0.0.1:8765

不要填完整路徑、不要加星號。設定剛完成可能需等待數分鐘才會生效。

此系統只需要 Client ID；Client secret 是伺服器端使用，不需要貼到網頁中。
